# Python Projects — Learning Workspace

This repository is a collection of small Python projects and exercises for learning and practice. Each project lives in its own folder and is intended to be self-contained, with simple instructions on how to run and test it.

## Current projects

- `CLI_calculator` — a simple command-line calculator implemented in `CLI_calculator/calculator.py`.

## Quick start

Prerequisites:

- Python 3.8 or newer
- (Recommended) Create and use a virtual environment for each project

Run the CLI calculator from the repository root:

```bash
python CLI_calculator/calculator.py
```

Or change into the project folder and run:

```bash
cd CLI_calculator
python calculator.py
```
